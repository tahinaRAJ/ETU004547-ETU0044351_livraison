Compréhension
Une société de livraison possède plusieurs véhicules, livreurs et zones de livraison.
Un livreur est affecté à un véhicule par jour.
Chaque livraison correspond à :
	•	un colis,
	•	une adresse de départ (entrepôt),
	•	une adresse de destination,
	•	un statut (en attente, livré, annulé).

Votre tâche est de créer la base de données, d’insérer des données de test, et d’effectuer les requêtes SQL nécessaires.

Travail à faire
Créer les tables et données nécessaires qui permettent de gérer les fonctionnalités :
Gestion des livraisons (Création , Statut)
les colis correspondants
Paramètre par livraison
cout de revient ( voiture + chauffeur (salaire journalier différent)
colis par kg
Créer directement les autres données nécessaires dans la table
Page pour afficher les bénéfices par période (jour, mois, année)
